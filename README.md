# 九划医教后台管理系统

医疗教育培训管理平台的后台管理系统。

## 技术栈

- **Vue** 2.5.21
- **Vue Router** 3.0.2
- **Vuex** 3.0.1
- **iView/View Design** 4.2.0
- **Element UI** 1.4.4
- **Axios** 0.18.0
- **Webpack** 4.28.1

## 环境要求

- Node.js >= 4.0.0
- npm >= 3.0.0

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发环境

```bash
# 启动开发服务器（热重载，默认 localhost:4000）
npm run dev
```

### 生产环境

```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm run server
```

### 其他命令

```bash
# 测试环境构建
npm run test

# 测试环境启动
npm start

# 门户环境构建
npm run portal

# 门户环境启动
npm run portal_start
```

## 产品流程

### 线上课流程

1. **创建产品** - 产品状态：编辑中
2. **添加讲师**（可选）- 产品状态：编辑中
3. **添加课程资料** - 启用课程资料（可选）- 产品状态：编辑中
4. **添加推荐资料** - 启用推荐资料（可选）- 产品状态：编辑中
5. **添加线上课**（可选）- 产品状态：编辑中
6. **产品信息** - 查看 → 添加 → 课程列表 - 产品状态：编辑中
7. **修改产品状态** - 产品状态：正常

### 线下课流程

1. **添加/复制学期**
2. **添加课程**
3. **启用线下课**
4. **发送学期** → 发送线下课

### 任务包流程

1. **添加分类**
2. **创建任务**
3. **激活任务** - 手动激活/定时激活

### 权限管理流程

1. **查看用户** → 编辑 → 分配权限
2. **创建学员**
3. **查看学员** → 查看详情 → 线上课 → 解锁测验
